# Project: Auth Service (Backend)

Python/FastAPI authentication microservice.

## Stack
- Python 3.12, FastAPI, SQLAlchemy
- PostgreSQL
- pytest for tests

## Conventions
- Routes in `app/routes/`
- Business logic in `app/services/`
- Exemplar: `app/routes/sessions.py`

## Commands
- `pytest` — run tests
- `ruff check .` — lint
